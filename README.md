# ChatKNML

## Getting started

### Embedding api

Download models:

```sh
cd models
git clone git@hf.co:intfloat/e5-large-v2
```

After starting the app, OpenAI-compatible embedding API will be available at: <http://172.16.3.101:5001/v1>

 Check the docs here: <http://172.16.3.101:5001/docs>

#### Starting app

```sh
docker compose up -d
```
